# 值得思考的点

1.  为什么人物移动的视角很奇怪，不是大多数的那种自由视角，视角和人物方向相互独立。本游戏视角只能有鼠标在一定的范围内浮动，而很大程度上去觉得人物行走的方向，也可能是地点，比如到了A区域就会切换视角，反正鼠标的移动带来的视角变换是比较小的。你想要转向某个视角，比如先朝着目标方向移动。不知道为什么这样设计，这样设计肯定是有意为之的，因为完全自由的视角肯定更简单一点。但是说实话我这个操作方式我真的感觉很别扭
2. 叙事顺序是打乱的，很有意思
3. 主角的父母对主角的态度是完全相反的，这也是设计好的，很多游戏的父母性格都不一样，这样才有冲突，才好看一些，如果都一样，那没必要用两个角色来表现，一个就够了，最好每个人有每个人的特点和设计的意义
4. 交互方式很有意思，特别是推动物体的时候，就是鼠标先往后拉，再往前推，模拟了体感
5. 故事排列成了一长串，顺序也被打乱了
6. 加入了一段音乐的情节，很感人
7. 进入游戏后是人脸，而且还会变换年纪，可以营造出惊喜感
