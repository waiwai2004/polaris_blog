# Polairs‘s Blog

这是给“歪歪 / Polairs‘s Blog”制作的粉色系个人博客与独立游戏项目展示页。

## 怎么打开

最简单的方法：

1. 下载 `polairs-blog-pretty-site.zip`
2. 解压压缩包
3. 双击里面的 `index.html`
4. 浏览器会直接打开网站

## 文件说明

- `index.html`：网站主文件，已经包含页面、样式和动画脚本
- `README.md`：说明文件
- `package.json`：给 Vercel / GitHub Pages 识别用的简单配置

## 怎么修改内容

用 VS Code 或记事本打开 `index.html`，搜索你想改的内容：

- `Polairs‘s Blog`
- `歪歪`
- `Forgecarft`
- `Echo from the Depth`
- `1556946930@qq.com`
- `19856308661`

改完保存，再双击 `index.html` 看效果。

## 怎么部署到 GitHub Pages

1. 新建一个 GitHub 仓库
2. 上传这些文件
3. 在仓库 Settings → Pages 中选择 Deploy from a branch
4. Branch 选择 `main`，目录选择 `/root`
5. 保存后等待 GitHub 生成访问链接

## 怎么部署到 Vercel

1. 打开 Vercel
2. 选择 Add New Project
3. 导入这个 GitHub 仓库
4. Framework Preset 选择 Other 或默认
5. Build Command 留空
6. Output Directory 留空
7. 点击 Deploy
